Kawaii Tetris
=============
Tetris written in Haskell using [FRP (functional reactive programming)][wiki].

Dependencies
------------
* [reactive-banana][banana] ([source][banana-src]), [reactive-banana-wx][banana] ([source][bananawx-src])
